Curso de JavaScript

El lenguaje de programacion que utilizan los navegadores para ejecutar aplicaciones web. Este curso es la base para aprender otros frameworks como Angular, React o Vue. Úsalo también para crear aplicaciones de servidor con NodeJs.

Curso de TypeScript

TypeScript es un lenguaje que extiende la sintaxis de JavaScript. Añade características nuevas en la programación orienta a objetos, y permite el tipado estricto. Esto permite detectar errores de código en tiempo de desarrollo, así como habilitar el autocompletado de código en todo momento.

Curso de Angular

Uno de los frameworks más utilizado para el desarrollo de aplicaciones web en el lado del cliente. Especialmente orientado a la creación de grandes aplicaciones, pero sirve perfectamente para crear cualquier tipo de aplicación web.

Curso de Css

Flex, grid, variables, rules, colores, unidades, gradientes, animaciones, nesting, etc.

Curso de Bootstrap

Bootstrap es un conjunto de herramientas de código abierto para diseño de sitios y aplicaciones web. Contiene plantillas de diseño con tipografía, formularios, botones, cuadros, menús de navegación y otros elementos de diseño basado en HTML y CSS, así como extensiones de JavaScript adicionales. A diferencia de muchos frameworks web, solo se ocupa del desarrollo front-end.

Curso de Tailwind

Tailwind CSS funciona revisando todos sus archivos HTML, componentes JS y otras plantillas en busca de clases, generando los estilos correspondientes y luego nos crae un archivo CSS estático.
Es rápido, flexible y confiable, sin tiempo de ejecución.
